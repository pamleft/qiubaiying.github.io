### 问题描述

给定一个由正整数组成且不存在重复数字的数组，找出和为给定目标正整数的组合的个数。

#### 示例 1：
```
nums = [1, 2, 3]
target = 4

所有可能的组合为：
(1, 1, 1, 1)
(1, 1, 2)
(1, 2, 1)
(1, 3)
(2, 1, 1)
(2, 2)
(3, 1)

请注意，顺序不同的序列被视作不同的组合。

因此输出为 7。
```

#### 分析：
动态规划  
dp[i] += dp[i-nums[j]] 其中i>coin[j]。  
```
func combinationSum4(nums []int, target int) int {
    dp := make([]int, target+1) 
    dp[0] = 1 
    for i := 0; i <= target; i++ {
        for j := 0; j < len(nums); j++ {
            if i - nums[j] >= 0 {
                dp[i] += dp[i-nums[j]]
            }
        }
    }
    return dp[target]
}
```
