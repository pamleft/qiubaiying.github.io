### 问题描述

给定一个二叉树，原地将它展开为链表。  

#### 示例 1:
```
输入：
    1
   / \
  2   5
 / \   \
3   4   6
输出：
1
 \
  2
   \
    3
     \
      4
       \
        5
         \
          6
```

#### 分析：
```
/**
 * Definition for a binary tree node.
 * type TreeNode struct {
 *     Val int
 *     Left *TreeNode
 *     Right *TreeNode
 * }
 */
func flatten(root *TreeNode)  {
    if root == nil {
        return
    }
    flatten(root.Left)
    flatten(root.Right)
    tmp := root.Right
    root.Right = root.Left
    root.Left = nil
    for root.Right != nil {
        root = root.Right
    }
    root.Right = tmp
}
```
