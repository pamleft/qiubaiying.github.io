### 问题描述

给定一个正整数，输出它的补数。补数是对该数的二进制表示取反。  

注意:  

给定的整数保证在32位带符号整数的范围内。  
你可以假定二进制数不包含前导零位。  

#### 示例 1:
```
输入: 5
输出: 2
解释: 5的二进制表示为101（没有前导零位），其补数为010。所以你需要输出2。
```
#### 示例 2:
```
输入: 1
输出: 0
解释: 1的二进制表示为1（没有前导零位），其补数为0。所以你需要输出0。
```
#### 分析：
每一位求反。   
代码
```
int findComplement(int num) {
    int b,i=1,sum=0;
	while(num){
		b=num%2;
		if(b==1)b=0;
		else b=1;
		b*=i;
		i*=2;
		num/=2;
		sum+=b;
	}
	return sum;
}
```
一次性求反
```
int findComplement(int num) {
    int mask = 1, temp = num;
         while(temp > 0) {
              mask = mask << 1;
             temp = temp >> 1;
         }
         return num^(mask-1);
}
```

位操作。
